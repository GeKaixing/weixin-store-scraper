## 解析要点

- **直播明细行组**: `标题 / 日期 / 时长 / 观看数 / 等 / N / 件` (7行一组)
- **短视频明细行组**: `标题 / 点赞数 / 分享数 / 喜欢数 / 带货商品` (4行一组)
- **购买力区间** 在 `购物偏好` 和 `直播带货` 之间, 四行一组: label/bounds/占/%
- **短视频tab切换**: 必须用 `page.mouse.click()` 模拟真实点击, dispatchEvent不生效
- **`document.body.innerText` 优于自定义 `getText()`**: 遍历 shadow DOM childNodes 的递归 `getText()` 在某些页面结构下只返回 1 行。用 `document.body.innerText` 替代，可获取完整128行页面文本。DETAIL_PARSE_JS 和短视频 evaluate 都需要改。
